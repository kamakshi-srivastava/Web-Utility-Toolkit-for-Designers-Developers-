AI Toolkit Website – Utilities for Creators/ Developers is a browser-based AI toolkit that helps creators, developers, and designers simplify their workflow through quick and easy-to-use tools built entirely with HTML, CSS, and JavaScript.

✨ Features Include:

  • 🔊 Text to Speech – Convert any text into speech using the Web Speech API.
  • 🖼️ Word Summarizer/Analyzer – Generate word summary and word analyzer.
  • 📄 Custom Lorem Ipsum – Create custom paragraphs of dummy text for mockups and layouts.

🛠️ Technologies Used:

  • HTML5, CSS3 (Flexbox, Grid)
  • JavaScript
  • Web Speech API
  • Several API's
